# Hinoki & Honey ☕

**Slow coffee. Quiet mornings.**

Hinoki & Honey is a beautifully crafted web experience for a boutique cafe and bakery located in Naka-Meguro, Tokyo. Designed with a calming "Japandi" (Japanese-Scandinavian) aesthetic, the platform utilizes soft earth tones, clean typography, and minimalist layouts to convey stillness, warmth, and artisanal care.

## ✨ Key Features

* **Japandi UI/UX:** A tranquil interface featuring warm earth tones, soft gradients, and a clean, minimalist design system.
* **Curated Digital Menu:** A high-quality visual grid showcasing signature coffees, matcha, and in-house baked pastries with detailed flavor profiles.
* **Seasonal Collections:** Dynamic feature sections highlighting limited-time seasonal offerings (e.g., Autumn Hojicha Maple, Yuzu Honey).
* **Interactive Map Integration:** Embedded OpenStreetMap component for seamless navigation to the physical storefront.
* **Responsive Layout:** A flawless browsing experience across all devices, maintaining aesthetic integrity on mobile, tablet, and desktop.

## 🛠️ Tech Stack

* **Framework:** React / TanStack Start
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **Package Manager:** Bun

## 🚀 Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine. This project uses **Bun** as the package manager.

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/bacuuyyy/hinoki-and-honey-cafe-website.git](https://github.com/bacuuyyy/hinoki-and-honey-cafe-website.git)

2. Navigate into the project directory:
   ```bash
   cd avoidwolves-esport-website

3. Install the dependencies:
   ```bash
   bun install

4. Start the development server:
   ```bash
   bun run dev

Open http://localhost:5173 (or the specific port shown in your terminal) with your browser to see the result.
